# DbBlobTool

O DbBlobTool é uma ferramenta desenvolvida em Java com SE que possibilita a otimização de imagens binárias armazenadas em bando de dados.

Suporte Atual: 
- MySQL
- SQL Server
- PostgreSQL
- SQLite
- Oracle
- MongoDB - pendente
